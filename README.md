# 🛡️ CVAT-Shield

Расширение для Chrome, которое защищает CVAT от случайных и злонамеренных действий.

## Возможности

- 🔒 Блокировка кнопок удаления проектов и задач
- 🎨 Скрытие Raw/Constructor с кнопкой показа
- 📷 Принудительное качество изображения 100%
- 👋 Приветствие пользователя 1 раз в день
- 🚫 Скрытие настроек проекта для обычных пользователей

## Установка

1. Скачайте репозиторий
2. Откройте `chrome://extensions/`
3. Включите "Режим разработчика"
4. Нажмите "Загрузить распакованное"
5. Выберите папку с расширением

## Совместимость

- Работает на `192.168.0.103:8080`
- CVAT версии 2.x
