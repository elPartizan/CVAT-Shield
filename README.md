# 🛡️ CVAT Shield

Расширение для Chrome, которое защищает CVAT от случайных удалений.

## Возможности

- 🔒 Блокировка кнопок удаления проектов, задач и лейблов
- 🎨 Скрытие Raw/Constructor с кнопкой показа
- 📷 Принудительное качество изображения 100%
- ⚙️ Скрытие настроек проекта

## Установка

1. Скачайте репозиторий
2. Откройте `chrome://extensions/`
3. Включите **"Режим разработчика"**
4. Нажмите **"Загрузить распакованное"**
5. Выберите папку с расширением

## Совместимость

- Работает на `192.168.0.103:8080`
- CVAT версии 2.x
